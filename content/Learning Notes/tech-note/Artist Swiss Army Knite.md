# 制作一个可以自动化帮助整理作品集、网站、申请的艺术家个人作品整理站

## 需求

- [ ]  同步网站内容 （容易的）
- [ ] 自动生成和同步作品集页面（难点在于：1. 如何生成一个符合pdf格式的文档，符合排版规范 2. 如何导出为pdf）
- [ ] 自定义作品集内容，但是仍然可以同步信息（难点：如何调整排版）
- [ ] AI自动整理作品信息（难点：是调用API呢还是自己训练模型呢？）
- [ ] 到期自动化提交申请 （难点：越过验证码，双重认证等问题）
